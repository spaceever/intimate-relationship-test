# 亲密关系测试 - 产品原型

## 项目说明

这是一个基于约翰·格雷《亲密关系情感依赖》理论的亲密关系测试产品原型。通过测试，用户可以了解自己的亲密关系模式类型：**焦虑型**、**安全型**或**回避型**。

## 原型页面

本原型包含以下8个关键页面：

1. **首页/引导页** - 产品介绍和开始入口
2. **说明页** - 测评说明和隐私声明
3. **测评页** - 题目卡片和Likert量表选择
4. **加载页** - 结果计算中的加载状态
5. **结果页-焦虑型** - 焦虑型用户的测试结果展示
6. **结果页-安全型** - 安全型用户的测试结果展示
7. **结果页-回避型** - 回避型用户的测试结果展示
8. **分享页** - 结果分享界面

## 使用方法

### 本地运行
1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # 或使用 Node.js
   npx serve .
   ```
3. 访问 `http://localhost:8000`

### 部署到 Vercel

#### 方法一：通过 Vercel 网站（推荐）

1. **准备项目**
   - 确保项目已包含 `index.html`、`vercel.json` 和 `package.json` 文件

2. **上传到 GitHub**
   - 在 GitHub 创建新仓库
   - 将项目文件推送上去：
     ```bash
     git init
     git add .
     git commit -m "Initial commit"
     git branch -M main
     git remote add origin <你的GitHub仓库地址>
     git push -u origin main
     ```

3. **在 Vercel 部署**
   - 访问 [vercel.com](https://vercel.com)
   - 使用 GitHub 账号登录
   - 点击 "Add New Project"
   - 选择你的 GitHub 仓库
   - Vercel 会自动检测配置，直接点击 "Deploy" 即可
   - 部署完成后会得到一个 `https://your-project.vercel.app` 的网址

#### 方法二：使用 Vercel CLI

1. **安装 Vercel CLI**
   ```bash
   npm install -g vercel
   ```

2. **登录 Vercel**
   ```bash
   vercel login
   ```

3. **部署项目**
   ```bash
   # 在项目目录下运行
   vercel
   
   # 生产环境部署
   vercel --prod
   ```

#### 方法三：通过 Vercel Dashboard 直接上传

1. 访问 [vercel.com/new](https://vercel.com/new)
2. 选择 "Upload" 选项
3. 直接将项目文件夹拖拽上传
4. 点击 "Deploy" 完成部署

### 更新部署

每次更新代码后，只需：
- 如果使用 GitHub：推送到仓库，Vercel 会自动重新部署
- 如果使用 CLI：运行 `vercel --prod` 重新部署
- 如果直接上传：重新上传文件即可

## 设计特点

- **移动端优先设计** - 最大宽度414px，适配手机屏幕
- **渐变色彩** - 使用紫色渐变作为主色调，不同类型使用不同颜色区分
- **交互反馈** - 选项点击有视觉反馈，按钮有悬停效果
- **进度指示** - 清晰显示测试进度
- **数据可视化** - 结果页面使用柱状图展示三维度得分

## 功能说明

### 测评流程
1. 用户进入首页，了解产品信息
2. 点击"开始测试"进入说明页，阅读测评说明和隐私声明
3. 勾选同意后进入测评页，逐题作答（Likert 1-5量表）
4. 完成所有题目后，进入加载页计算结果
5. 跳转到对应的结果页面，展示类型、特点和建议
6. 可选择分享结果或获取个性化建议

### 类型判定
- **焦虑型**：焦虑维度得分最高（≥50分）
- **安全型**：安全维度得分最高（≥70分）且焦虑和回避都较低
- **回避型**：回避维度得分最高（≥50分）

### 结果展示
每个结果页面包含：
- 类型徽章标识
- 三维度得分柱状图（焦虑、回避、安全）
- 该类型的特点列表（4条）
- 个性化建议列表（4条）
- 操作按钮（获取建议、分享、了解更多）
